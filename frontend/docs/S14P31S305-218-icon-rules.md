# S14P31S305-218 아이콘 규칙 정의

## 1. 목적
- 에너지/설비/이벤트 아이콘 규칙을 통일한다.

## 2. 스타일
- 아이콘 스타일: `outlined`
- 크기: 16 / 20 / 24px
- 선 굵기: 일관 유지

## 3. 카테고리
- 에너지: 태양광, ESS, 디젤, 그리드
- 설비: 스위치, 모선, 부하센터, 게이트웨이
- 이벤트: 알람, 명령 결과, 시스템 상태

## 4. 상태 반영
- Normal: 기본 컬러
- Warning: 경고 컬러
- Critical: 심각 컬러 + 배지 강조
- Offline: 비활성 컬러

## 5. 우측 상단 아이콘 트리거 규칙

| 아이콘 | 동작 | 우측 패널 모드 |
|---|---|---|
| Bell | 알람 목록 오픈/닫기 | Alarm Panel |
| List | 최근 명령 결과 오픈/닫기 | Recent Command Panel |
| Globe | 국가/언어 선택 오픈/닫기 | Country/Language Panel |
| Gear | 설정 오픈/닫기 | Settings Panel |

> 동시에 여러 패널을 열지 않는다. 새 아이콘 클릭 시 기존 모드에서 전환한다.

## 6. 배지/활성 상태 규칙

- 배지 숫자: 미확인 건수 표시
- 활성 아이콘: `color.primary` 보더/글로우
- Critical 알람 존재 시 Bell 배지를 우선 강조
