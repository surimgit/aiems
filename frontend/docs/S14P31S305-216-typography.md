# S14P31S305-216 타이포그래피 정의

## 1. 목적
- 관제 화면 가독성을 위한 폰트/크기/행간 규칙을 고정한다.

## 2. 기본 규칙
- Font Family: `Pretendard`
- 숫자 KPI: 시각적 정렬이 필요한 경우 tabular 숫자 사용

## 3. 타입 스케일
| Style | Size | Weight | Line-height | 사용처 |
|---|---:|---:|---:|---|
| `display-lg` | 32 | 700 | 40 | 메인 제목 |
| `heading-md` | 24 | 700 | 32 | 섹션 제목 |
| `title-sm` | 18 | 600 | 24 | 카드 제목 |
| `body-md` | 14 | 500 | 20 | 기본 본문 |
| `caption` | 12 | 500 | 16 | 보조 텍스트 |

## 4. 수치 표기
- 전력: 정수 + `kW`
- 에너지: 천단위 콤마 + `kWh`
- 비율: 정수 + `%`
- 시간: `HH:mm:ss (KST)`

## 5. 우측 패널 타이포 계층

| 영역 | 권장 스타일 | 비고 |
|---|---|---|
| 패널 타이틀 | `title-sm` | 예: 알람(Top 3), 국가 선택 |
| 섹션 헤더 | `body-md` (600) | 예: 간단 제어, 상세 제어 |
| 리스트 본문 | `body-md` | 예: 명령 결과/알람 항목 |
| 보조 정보 | `caption` | 예: 시각, 단위, 보조 텍스트 |

## 6. 축소 모드 가독성 규칙

- 우측 패널 오픈으로 메인 영역이 줄어들 때도 핵심 값(`kW`, `%`, 상태 텍스트)은 유지한다.
- 카드 내부 보조 문구는 축약 가능하지만 상태/수치 라인은 축약하지 않는다.
- 시간/보조 라벨은 `caption` 유지, 최소 12px 미만 사용 금지.
